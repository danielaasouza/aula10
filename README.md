# Como compilar

 Linux / Mac
```bash
gcc exercicio1.c -o exercicio1
./exercicio1

gcc exercicio2.c -o exercicio2
./exercicio2

gcc exercicio3.c -o exercicio3
./exercicio3

```

Windows
```bash
gcc exercicio1.c -o exercicio1.exe
exercicio1.exe

gcc exercicio2.c -o exercicio2.exe
exercicio2.exe

gcc exercicio3.c -o exercicio3.exe
exercicio3.exe
